About Us   Our Project

This project created by Purple Rookies accepts a chest X-Ray and identifies if a patient has pneumonia with accuracy of 90.22%. The programming language used to create this project was Python. Many scientific, ML and computational libraries (e.g. TensorFlow, scikit-learn, NumPy, pandas, matplotlib) were used as well. The ML algorithm used for classification was Convolutional Neural Network (CNN) model. Our team has worked together to create this project with very little or no prior experience in using the software tools mentioned above.
